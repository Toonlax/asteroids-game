# Asteroids Game
A simple Asteroids-style game built with Python and pygame as part of the Boot.dev curriculum.

## Controls
- W — Move forward
- S — Move backward
- A — Rotate ship counter-clockwise
- D — Rotate ship clockwise
- Space — Shoot

## Gameplay
- Asteroids spawn and move across the screen
- Large asteroids split into smaller ones when destroyed
- Colliding with an asteroid ends the game

## Requirements
- Python 3.x
- pygame

## How to Run
```bash
python main.py
